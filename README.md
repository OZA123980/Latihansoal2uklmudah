# Latihansoal2uklmudah
Latihan UKL 1
Program ini berfungsi untuk menentukan apakah suatu bilangan/angka yang dimasukkan pengguna adalah bilangan ganjil atau genap.
Cara kerjannya:
-Pengguna memasukkan sebuah bilangan bulat.
-Program memeriksa sisa hasil bagi bilangan tersebut dengan 2(bilangan %2).
-Jika sisa bagi = 0 - bilangan genap,
-jika tidak = 0 - bilangan ganjil.
-Program menampilkan hasilnya ke layar yang telah dimasukkan oleh pengguna bilangan genap atau ganjil.
